ติดตั้งเครื่องมือบนเครื่อง

15 นาที

ติดตั้ง 3 ตัว: Node.js (จำเป็น), VS Code (แก้โค้ด), Git (สำหรับ deploy)

ANode.js (จำเป็น)

  1. เปิด nodejs.org
  2. กดโหลด LTS (เลขเวอร์ชั่น 20 หรือ 22)
  3. เปิดไฟล์ .msi → Next → Next → Install (default ทั้งหมด)
  4. เช็คว่าติดตั้งสำเร็จ — เปิด PowerShell แล้วพิมพ์:
powershellnode --version
npm --version

ถ้าขึ้นเลขเวอร์ชั่น = ติดตั้งสำเร็จ ✅

BVS Code (แนะนำ)

  1. โหลดจาก code.visualstudio.com
  2. ติดตั้งตาม default

CGit (จำเป็นสำหรับ deploy)

  1. โหลดจาก git-scm.com/download/win
  2. ติดตั้งตาม default ทั้งหมด — ห้ามเปลี่ยน option

4

สร้าง Service Account (ส่วนยากสุด)

15 นาที

Service Account = "บัญชีหุ่นยนต์" ที่ให้เว็บอ่าน Sheet ได้ โดยไม่ต้องเปิด Sheet ให้ public

Aสร้าง Google Cloud Project

  1. เข้า console.cloud.google.com (ล็อกอินอีเมลเดียวกับ Sheet)
  2. ครั้งแรกจะมี popup ให้ยอมรับ Terms → กด Agree and Continue
  3. มุมบนซ้าย กดที่ชื่อ project (หรือ "Select a project") → NEW PROJECT
  4. กรอก:
    • Project name: TSAK Web
    • Location: ปล่อย default
  5. กด CREATE → รอ 10 วินาที
  6. รอจน notification ขวาบนแจ้ง "Project created" → กด SELECT PROJECT

Bเปิดใช้งาน Google Sheets API

  1. ในแถบ search ด้านบน พิมพ์: Google Sheets API
  2. คลิกผลลัพธ์แรก
  3. กดปุ่มน้ำเงิน ENABLE → รอ 10 วินาที

Cสร้าง Service Account

  1. เมนูซ้ายบน (☰) → APIs & Services → Credentials
  2. ด้านบน กด + CREATE CREDENTIALS → Service account
  3. กรอก:
    • Service account name: tsak-sheet-reader
    • (อื่นๆ ปล่อย default)
  4. กด CREATE AND CONTINUE
  5. ข้าม 2 ช่องถัดไป (Role และ Grant access) → กด DONE

Dดาวน์โหลด Key (JSON)

  1. ในหน้า Credentials จะเห็น service account ที่เพิ่งสร้าง → คลิกที่ชื่อมัน
  2. แท็บด้านบนกด KEYS
  3. กด ADD KEY → Create new key
  4. เลือก JSON → กด CREATE
  5. ไฟล์ .json จะถูกดาวน์โหลดอัตโนมัติ

เก็บไฟล์ JSON นี้ให้ดี ห้ามทำหาย ห้าม commit ขึ้น git ห้ามแชร์ใคร — เพราะเป็น "กุญแจ" เข้าระบบ

EShare Sheet ให้ Service Account

  1. เปิดไฟล์ JSON ด้วย Notepad
  2. หาบรรทัด "client_email": "tsak-sheet-reader@...iam.gserviceaccount.com" → copy email นี้
  3. กลับไปที่ Google Sheet ที่สร้างไว้ขั้น 3
  4. กดปุ่ม Share มุมขวาบน
  5. วาง email ที่ copy → เปลี่ยนสิทธิ์เป็น Viewer → กด Send
  6. ถ้ามี popup ถามว่าจะส่ง email — กด "Share anyway"
5

เชื่อม Sheet เข้ากับเว็บ

5 นาที

Aสร้างไฟล์ .env.local

  1. ใน VS Code คลิกขวาที่พื้นที่ว่างใน File Explorer ซ้ายมือ → New File
  2. ตั้งชื่อ: .env.local (มีจุดข้างหน้าด้วย)

Bกรอกค่า 3 ตัว

เปิดไฟล์ JSON ที่ดาวน์โหลด แล้ว copy ค่ามาใส่:

.env.localGOOGLE_CLIENT_EMAIL="tsak-sheet-reader@xxxxx.iam.gserviceaccount.com"
GOOGLE_PRIVATE_KEY="-----BEGIN PRIVATE KEY-----\nMIIEv...ยาวมาก...\n-----END PRIVATE KEY-----\n"
GOOGLE_SHEET_ID="1AbCdEfGhIjKlMn123456789"

จุดสำคัญ private_key: ใน JSON จะเป็น "private_key": "-----BEGIN...\n...\n-----END...\n" — copy ทั้งก้อน (รวม \n ทั้งหมด) วางใน quote "" โค้ดจะแปลง \n เป็น newline จริงให้เอง

Cรันเว็บใหม่

ถ้า dev server ยังรันอยู่ → กด Ctrl + C → แล้ว:

powershellnpm run dev

เปิด localhost:3000 → ตัวเลขสถิติและการ์ดกิจกรรมจะขึ้นจาก Google Sheet แล้ว 🎉

Dทดสอบ live update

แก้ตัวเลขใน Sheet (เช่น members จาก 58 → 100) → กลับมา refresh เว็บ → ตัวเลขเปลี่ยน

6

ขึ้นโค้ดบน GitHub

10 นาที

Aสมัคร GitHub (ถ้ายังไม่มี)

เข้า github.com → Sign up ด้วยอีเมลกลางชมรม

Bสร้าง Repository

  1. มุมบนขวา กด + → New repository
  2. ตั้งชื่อ: tsak-web
  3. เลือก Private
  4. ห้ามเช็คอะไรเลย (ไม่ต้อง README/gitignore เพราะเรามีแล้ว)
  5. กด Create repository

ต้องเป็น Private — กัน Google credentials รั่ว แม้ว่า .gitignore จะกันไว้แล้ว

CPush โค้ดขึ้น GitHub

ใน terminal ของ VS Code (โฟลเดอร์โปรเจกต์):

powershellgit init
git add .
git commit -m "Initial commit: TSAK web base setup"
git branch -M main
git remote add origin https://github.com/<your-username>/tsak-web.git
git push -u origin main

(แก้ <your-username> เป็น username GitHub ของคุณ)

ไฟล์ .env.local จะไม่ถูก push ขึ้น เพราะ .gitignore กันไว้แล้ว ✅

7

Deploy บน Vercel (ฟรี)

10 นาที

Aสมัคร Vercel

  1. เข้า vercel.com/signup
  2. เลือก Continue with GitHub → อนุญาตเชื่อมต่อ

BImport Project

  1. หน้า dashboard กด Add New... → Project
  2. เลือก repository tsak-web → Import
  3. หน้า config: ปล่อย default ทุกอย่าง (Vercel จะรู้ว่าเป็น Next.js เอง)

Cใส่ Environment Variables

เลื่อนลงไปที่ Environment Variables ใส่ 3 ตัว (จาก .env.local):

NameValue
GOOGLE_CLIENT_EMAIL(copy จาก .env.local)
GOOGLE_PRIVATE_KEY(copy จาก .env.local รวม \n)
GOOGLE_SHEET_ID(copy จาก .env.local)

DDeploy

  1. กด Deploy → รอประมาณ 1–2 นาที
  2. เสร็จแล้วจะเห็นหน้า "Congratulations!" + URL
  3. ได้ลิงก์เว็บประมาณ https://tsak-web.vercel.app ✨

🎊 จบแล้ว!

ตอนนี้คุณมีเว็บออนไลน์ ฟรี 100% รุ่นน้องแก้ข้อมูลผ่าน Google Sheet ได้เลย และทุกครั้งที่ git push Vercel จะ deploy ใหม่อัตโนมัติ
